# Kare Matris Çarpımı 🧮

Java Swing kullanılarak geliştirilmiş, görsel arayüze sahip bir kare matris çarpım uygulamasıdır. 

## 🚀 Özellikler

* **Dinamik Boyutlandırma:** İstenilen boyutta (NxN) kare matris oluşturabilme.
* **Esnek Veri Girişi:** Matris değerlerini arayüz üzerinden tek tek elle girme veya rastgele sayılarla otomatik doldurma seçeneği.
* **Görselleştirilmiş Sonuç:** Matris çarpım sonucunu anlık olarak ekranda görüntüleme.
* **Geriye Dönük Kontrol:** Hesaplama yapıldıktan sonra, girilen orijinal matrisleri ayrı bir pencerede yan yana inceleyebilme.


## 💡 Geliştirici Notu

Bu proje benim ilk denememdir. Kodlama ve geliştirme sürecinde yapay zeka asistanı olarak Gemini 3.1 Pro kullanılmıştır.
